### Data
For å generere denne nettsiden, brukes json-filer med data. Dette gjøres da det er lettere å gjøre endringer her enn å ha alt statisk hardcodet i selve koden. Json-filene kan sees på som `api` for frontenden, men fremfor å hente data fra eksterne kilder, er all data lagret lokalt.
